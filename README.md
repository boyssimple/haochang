# haochang
# 仿好唱APP
基于XMPP聊天APP-IOS(OC)项目，构架、接口处理、主页开发、音乐播放
## 安装说明
1. pod isntall
2. 服务器端安装openfire
3. AtChat.pch文件中IP地址改为服务器地址
## 功能
1. 首页接口
2. 音乐播放
